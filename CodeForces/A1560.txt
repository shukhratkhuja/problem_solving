A. Нелюбовь к тройкам
ограничение по времени на тест1 секунда
ограничение по памяти на тест256 мегабайт
вводстандартный ввод
выводстандартный вывод
Поликарп не любит целые числа, которые делятся на 3
 или оканчиваются цифрой 3
 в десятичной записи. Числа, для которых выполняются оба условия одновременно, Поликарп тоже не любит.

Поликарп начал выписывать последовательно целые положительные (большие, чем 0
) числа, к которым он относится хорошо: 1,2,4,5,7,8,10,11,14,16,…
. Выведите k
-й элемент из этой последовательности (элементы нумеруются с 1
).

Входные данные
В первой строке записано одно целое число t
 (1≤t≤100
) — количество наборов входных данных. Далее следуют t
 наборов входных данных.

Каждый набор входных данных состоит из одной строки, содержащей одно целое число k
 (1≤k≤1000
).

Выходные данные
Для каждого набора входных данных в отдельной строке выведите одно число x
 — k
-й элемент в последовательности, выписанной Поликарпом.
